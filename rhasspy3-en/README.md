# Rhasspy 3 (en)

English add-on with [Rhasspy 3](https://github.com/rhasspy/rhasspy3).

After installing, visit http://homeassistant.local:13331

To enable "insecure" microphone access in Chrome:

1. Visit chrome://flags/#unsafely-treat-insecure-origin-as-secure in Chrome
2. Add http://homeassistant.local:13331
3. Click the Relaunch button
